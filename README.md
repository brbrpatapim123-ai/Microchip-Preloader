# Проект "Microchip-Preloader"

## Описание проекта
Данный проект создан для объяснения как пользоваться приложением GitHub Desktop. 

## Цели проекта
- Помощь новичкам в опубликовке своего проекта
- Пояснение что такое GitHub Desktop
- Как использовать приложение

## Используемые технологии
- Index.html для создания сайта с анимацией микрочипа.
- Style.css для анимации этого микрочипа.

## Установка зависимостей
Зависимость отсутствует

## Пример запуска
Скачайте файлы и запустите index.html.
