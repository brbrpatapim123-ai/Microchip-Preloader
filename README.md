# Проект "GitHub Desktop"

## Описание проекта
Данный проект создан для объяснения как пользоваться приложением GitHub Desktop. 

## Цели проекта
- Помощь новичкам в опубликовке своего проекта
- Пояснение что такое GitHub Desktop
- Как использовать приложение

## Используемые технологии
- Index.html для создания сайта с анимацией микрочипа.
- Style.css для анимации этого микрочипа.

## Установка зависимостей
Зависимость отсутствует

## Пример запуска
Для запуска сайта у вас уже должен быть установлен Visual studio code.

Для того чтобы открыть сайт вам нужно скопировать код для index.html и style.css.

Затем создать папку и открыть Visual studio code в приложении нажать на кнопку file, потом Open Folder выбирайте свою папку, далее просто добавьте файлы (index.html и style.css)

Вставте нужный код в нужный файл и запустите index.html из созданной папки.

Проект выполнен в образовательных целях на онлайн-курсе "Основы Python" школы "Лидер".
